# Input parameters 
输入参数

If this actor is run on the [Apify platform](https://console.apify.com/), our simple interface should help you out in configuring all the parameters. This scraper follows the following

- **profiles** - scrapes TikTok profiles. You can scrape one or a few profiles at the same time.
- **hashtags** - scrapes TikTok for hashtags. You can scrape one or several hashtags in one run.
- **postURLs**scrapes specific TikTok posts. You can scrape one or several posts simultaneously.
- **resultsPerPage** - sets up the amount of results you expect to be scraped. Can be from 10 to thousands of results, so a thousand profiles with all the surrounding info, for instance.
- **maxRequestRetries** - indicates the limit of scraper retries in case of any errors.
- **proxyConfiguration** - sets up the proxies.

如果这个actor在[Apify平台](https://console.apify.com/)上运行，我们简单的界面应该可以帮助您配置所有参数。 该抓取工具遵循以下

- **个人资料** - 抓取 TikTok 个人资料。 您可以同时抓取一个或多个用户。
- **主题标签** - 抓取 TikTok 的主题标签。 您可以一次抓取一个或多个主题标签。
- **postURLs** 抓取特定的 TikTok 帖子。 您可以同时抓取一个或多个帖子。
- **resultsPerPage** - 设置您希望抓取的结果数量。 可以是 10 到数千个结果，例如包含所有周围信息的一千个用户。
